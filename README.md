# TalkMateAI
A Spoken English Learning ChatBot 

Application Type:- AI-assisted grammar correction, vocabulary enhancement, and spoken English support platform

Abstract:-TalkMate AI is a web-based spoken English learning chatbot designed to improve a learner’s ability to write and speak grammatically correct English. The application allows users to type a sentence, submit it for grammar analysis, receive a corrected version, view a concise explanation of detected errors, explore vocabulary suggestions, and listen to audio pronunciation of the corrected sentence. The platform is especially useful for learners who need quick, interactive, and easy-to-understand feedback rather than static grammar rules.
The backend application is implemented using Python and FastAPI and includes request validation through Pydantic models, grammar correction through the LanguageTool API, vocabulary extraction and enrichment through spaCy, NLTK WordNet, and TextBlob, and audio generation through gTTS. The frontend is built using HTML, CSS, and JavaScript, and provides a responsive user interface featuring sentence input, correction display, vocabulary cards, API health monitoring, and audio playback. 

Problem statement:-English is one of the most widely used languages in education, technology, business, and international communication. However, many students and beginner users face difficulty in sentence construction, grammar correctness, vocabulary selection, and confident expression. Traditional learning methods often involve textbooks, classroom explanation, or passive video content, which do not always provide immediate personalized correction.

Solution:-A chatbot-based learning assistant can solve this issue by enabling learners to interact with a smart system that checks grammar, suggests improvements, recommends vocabulary, and generates audio output for better spoken understanding. Unlike a static grammar guide, such a system can respond instantly to a learner’s own sentence and thereby make the learning process practical and engaging.
